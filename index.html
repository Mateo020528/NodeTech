Aprende sobre media queries.

El diseño responsivo tiene como objetivo adaptar la visualización de una página web a cada dispositivo de la mejor manera posible. Las Media Queries se utilizan para obtener las características técnicas del dispositivo que solicita la visualización de la página web.

Aprende sobre display grid.

El internet móvil presenta un gran número de desafíos para los diseñadores de páginas web: debido a la enorme variedad de diseños de los dispositivos móviles, es imposible saber qué formato tiene la pantalla en la que se visualizará el contenido web.

Aprende sobre apis

El término API significa Application Programming Interface y se basa en un conjunto de definiciones y protocolos que tienen el propósito de integrar sistemas y facilitar la comunicación entre aplicaciones de software según una serie de reglas.
